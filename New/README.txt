Pour lancer la base de données : 

taper : mongod --dbpath /backend/data

Pour lancer le serveur aller dans /backend/

taper : npm start 

Pour lancer le frontEnd aller dans /frontend/frontend/src/app/

taper : ng serve --open


Les modules necessaires a l'execution sont listés dans package.json (il y en a deux, dans le frontend et dans le backend)
